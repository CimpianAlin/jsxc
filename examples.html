Usage Examples

ownCloud Integration

ownCloud already provides self-hosted storage and sharing of files, contacts, calendars, and much more. Collaborating on documents frequently requires coordination. This is where JSXC (in the shape of the ojsxc app for ownCloud) caters for your needs.

Besides the standard features of JSXC, the ownCloud integration adds:

  • Unified login: Logging into ownCloud also logs you in to JSXC/XMPP
  • Configuration from within ownCloud's administrative interface

SOGo Integration

Scalable Open Groupware, SOGo combines a Thunderbird-lookalike web interface for email browsing with enterprise-quality CalDAV/CardDAV calendaring and address books. JSXC not only shares web browser real estate with SOGo, but actually augments it. Having received an email, you can immediately call back via instant messaging or XMPP-based VoIP audio/video call, if you require additional information or if the matter is better settled face-to-face. The standard features of JSXC are extended by:

  • Unified login: Logging into SOGo also logs you in to JSXC/XMPP
  • JSXC recognizes XMPP IDs (Jabber ID, JID) in the SOGo content (address books, emails, â€¦)
  • JIDs in and enhances them with presence information (the colored bullets in the screen shot's magnifying glass).
