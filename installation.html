Installation

Prerequisites

  • Install and configure an XMPP server with BOSH support, e.g. ejabberd
  • Make sure your XMPP server's BOSH URL is reachable from JavaScript files on the web server (same origin policy, SOP). [Configuration]

JSXC with ownCloud

To use JSXC with ownCloud, follow the instructions here or install from ownCloud Apps Repository

JSXC with anything else

Download JSXC from GitHub, configure it and enable login hooks, if desired. This is currently only recommended for experts.
